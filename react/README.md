### `check`

`tsc`・`prettier`・`eslint`・`prettier` を用いた静的解析を**並列**で実行する。

```bash
npm run check
```

### `format`

`prettier`・`eslint`・`prettier` を用いたフォーマットを**並列**で実行する。

```bash
npm run format
```
